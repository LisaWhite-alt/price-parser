# price-parser

## Описание проекта

Сайт для приложения, выявляющего несоответствия цен на сайтах дилеров по отношению к рекомендуемому прайсу.

### Технологии

* Python 3.8.5
* Flask 2.0.2
* uWSGI
* Nginx
